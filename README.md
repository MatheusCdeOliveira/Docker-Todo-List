<details>
  <summary><strong>🛠 Como rodar o projeto</strong></summary><br />

 1 - Clone o repositório
  
 2 - Entre na pasta Docker:
   - `cd docker`
  
 3 - Rode o container:
   - `docker-compose up -d`
  
 4 - Veja a aplicação rodando:
   - `http://localhost:3000/`
 
</details>
